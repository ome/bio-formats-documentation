IPLab
=====

Extensions: .ipl

Developer: `Scanalytics <http://scanalytics.com/>`_

Owner: `BD Biosystems <http://www.bdbiosciences.com/>`_

Support
-------

SCIFIO: |no|

Export: |no|

Supported Metadata Fields: `IPLab Reader <http://dev.loci.wisc.edu/trac/software/wiki/IPLabReader>`_

Freely Available Software: `IPLab Reader plugin for ImageJ <http://rsb.info.nih.gov/ij/plugins/iplab-reader.html>`_

We currently have: 

-  An IPLab specification document (v3.6.5, from 2004 Dec 1 in PDF)
-  Several IPLab datasets

We'd like to have: 

-  More IPLab datasets (32-bit integer or floating point data)
