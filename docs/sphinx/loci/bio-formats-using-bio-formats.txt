Using Bio-Formats
=================

--------------

Â 

Bio-Formats is a Java-based library that converts `over 115 microscopy
formats <http://www.loci.wisc.edu/bio-formats/formats>`_ into the
OME-XML and OME-TIFF formats. Bio-Formats can be used with numerous
`applications <http://www.loci.wisc.edu/bio-formats/applications>`_ but
is most commonly used with ImageJ and FIJI.

The purpose of this document is to explain Bio-Formats's features and
how to use it within ImageJ and FIJI. The sections include:

-  `Installing Bio-Formats in ImageJ <installing-bio-formats-imagej>`_
-  `Bio-Formats Features <bio-formats-features>`_
-  `Loading Images with Bio-Formats <using-bio-formats-load-images>`_
-  `Managing Memory Using
   Bio-Formats <managing-memory-using-bio-formats>`_
-  `Reporting a Bug <bug-reporting>`_
