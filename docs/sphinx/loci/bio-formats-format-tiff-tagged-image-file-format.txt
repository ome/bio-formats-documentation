TIFF (Tagged Image File Format)
===============================

--------------

Extensions: 

.tif

Developer: 

Aldus and Microsoft

Owner: 

`Adobe <http://www.adobe.com>`_

Support

Export: |yes|

Supported Metadata Fields: 

`TIFF Reader <http://dev.loci.wisc.edu/trac/software/wiki/TiffReader>`_

Sample Datasets: 

`LZW TIFF data
gallery <http://marlin.life.utsa.edu/Data%20Gallery.html>`_

Sample Datasets: 

`Big TIFF <http://tiffcentral.com/>`_

We currently have: 

-  A `TIFF specification
   document <http://partners.adobe.com/asn/developer/PDFS/TN/TIFF6.pdf>`_
   (v6.0, from 1992 Jun 3, in PDF)
-  A large number of TIFF datasets
-  A few BigTIFF datasets

We'd like to have: 

More TIFF datasets:

-  anything unusual—preferably rare/unsupported compression or
   photometric interpretation type

   GB).
