Micro-Manager
=============

- Micromanager offers the functionality for saving the acquired images in tiff/ome-tiff format. 
- Based on the mode of saving and the configuration settings, the acquired image can be saved 
  with/without a companion file (metadata.txt).

    :align: center

| ``*``  : minimal metadata : Minimum amount of metadata required to validate an ome-xml.

| Micro-Manager --> Options --> Select : Create metadata.txt file with Image Stack Files
